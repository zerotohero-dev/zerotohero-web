About VCF

  • VCF is a full-stack Infrastructure as a Service (IaaS) platform that delivers software-defined compute, storage, networking, security, and management.
  • Integrated automation enables a self-service platform to deploy VMs and containers for developer agility.
  • Hardened platform offering built-in resilience, scaling, and clustering for non-stop operations.
  • It provides automation and orchestration to simplify Day 0, Day 1, and Day 2 tasks.
  • It simplifies full stack deployment.

The purpose of VCF is to design, plan, deploy, and manage your SDDC by using automation capabilities.

VMware Cloud Foundation is a unified private-cloud platform that combines the scale and agility of public cloud with security and performance of private cloud. delivering increased productivity and lower Total Cost of Ownership (TCO).

VCF Components

VCF consists of the following parts:

Core Infrastructure

These are the heavy-lifters of VCF.

  • vSphere (compute and VM runtime)
  • vSAN (storage)
  • NSX (networking and security)

Management Plane

Management Plane is about platform control and lifecycle management.

  • VCF Operations (visibility, health, and optimization)
  • VCF Automation (provisioning, policy, and workflows)

Enablement Plane

These are meta components. They neither run workloads, nor operate them directly. They enable installation, integration, and extensibility.

  • VCF Installer
  • VCF SDKs, APIs, and CLIs

Other Related Services

  • VMware Live Recovery
  • VMware Data Services Manager
  • VCF Business Services Console
  • VCF Usage Meter
  • VMware Cloud
  • VMware Skyline
  • Cloud Services Console
  • Gated SDK

Security and Compliance

  • Governance
  • STIG
  • VCF Roles and Permissions
  • VCF Single Sign-On
  • VCF Identity Broker

VCF Capabilities

VCF builds, manages, operates, secures, monitors, and govern an entire Private Cloud.

Build

  • Deploy a new VCF platform
  • Converge existing virtual infrastructure to a new VCF platform.
  • Deploy a Workload Domain
  • Import an existing vCenter instance to a Workload Domain.
  • Add a vSphere Cluster or an ESX host to a Workload Domain.
  • Configure vSAN Stretched Cluster
  • Manage network connectivity using NSX Edge Clusters and Transit Gateways in vCenter
  • Manage virtual infrastructure in vCenter

Manage (VCF Automation)

  • Licensing
  • Configure IAM and SSO
  • Link vCenter instances for centralized inventory management
  • Manage certificates across the VCF Fleet
  • Manage password across the VCF Fleet
  • Manage tags across the VCF Fleet
  • Manage the configuration settings of vCenter instances and vSphere Clusters.
  • Backup and restore the VCF Fleet and its managed VCF Instances
  • Migrate workloads across Workload Domains and vCenter instances

Operate (VCF Operations)

  • Deploy and configure vSphere Supervisor for running Kubernetes Workloads and VM Workloads
  • Licensing
  • Configure alerts
  • Perform log analysis and troubleshooting
  • Monitor network operations
  • Monitor storage operations
  • Perform infrastructure diagnostics and VCF health monitoring
  • Configure and use monitoring dashboard and monitoring reports
  • Workload monitoring and observability
  • Manage cost and capacity
  • Mitigate security risks
  • Enforce security compliance
  • Manage infrastructure, identity, and access across consumer organizations

Network and Storage

  • Manage vSAN storage
  • Manage networking in NSX

Platform Operations (VCF Operations)

  • Link vCenter instances for centralized inventory management
  • Manage certificates across the VCF Fleet
  • Manage passwords across the VCF Fleet
  • Manage tags across the VCF Fleet
  • Manage the configuration settings of vCenter instances and vSphere Clusters.

Organization Administration

  • Prepare the collection of VM-base workload Blueprints and infrastructure services for the organization and end users

DevOps

  • Provision VMs
  • Provision Kubernetes clusters
  • Provision vSphere Pods
  • Provision persistent volumes
  • Provision secret stores
  • Install container registries
